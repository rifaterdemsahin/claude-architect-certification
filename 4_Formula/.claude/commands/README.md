# ⚡ commands — Custom Slash Commands

> **Purpose:** Custom slash commands available to Claude Code for common development workflows.

## Files

| Command | File | Purpose |
|---------|------|---------|
| `/log-error` | `log-error.md` | Append entry to `6_Semblance/error.log` |
| `/log-fix` | `log-fix.md` | Append entry to `6_Semblance/fix.log` |
| `/stage-commit` | `stage-commit.md` | Stage, commit, and push current changes |
| `/retrospective` | `retrospective.md` | Write milestone entry to `lessons_learned.md` |
| `/init-project` | `init-project.md` | Bootstrap all 7 stage folders for a new project |

## Rules
- Each command file must be a self-contained markdown with clear instructions
- Keep commands generic enough for use across different projects
