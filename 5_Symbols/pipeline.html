🧩 Production Pipeline

End-to-end course production workflow — from problem definition to published content

Pre-Production Production Post-Production Publishing

🛠 Each stage lists the tools used. Chips marked tool are estimated — likely candidates where the exact tool isn't pinned down.

❓
01 Problem Pre-Production
Identify and articulate the core problem the course will solve. What gap exists in the market? Who is affected and why does it matter?
📥 Input
  • Market research & gap analysis
  • Learner pain points & feedback
  • Industry trends & technology shifts
  • Personal observations & experience
📤 Output
  • Clear problem statement document
  • Target audience definition
  • Value proposition hypothesis
  • Success criteria & KPIs
🛠 Tools Used
🤖 Claude 🔍 Perplexity 🗄️ Supabase 🪨 Obsidian 💬 Reddit / forums
→ Open Problem page
⬇
🏠
02 Solution (Product) Pre-Production
Define the course as the solution. Articulate what learners will be able to do after completing the course. This frames the entire production effort.
📥 Input
  • Problem statement
  • Target audience profile
  • Competitive course analysis
  • Available expertise & resources
📤 Output
  • Course concept & positioning
  • Learning objectives (Bloom's taxonomy)
  • Course format & delivery method
  • Preliminary module breakdown
🛠 Tools Used
🤖 Claude 📝 Markdown 🪨 Obsidian 🎨 Canva 📐 Bloom's taxonomy template
→ Open Home page
⬇
🔬
03 Research Pre-Production
Deep-dive into the subject matter while self-learning the content. Record draft takes (screen captures, walkthroughs) as you explore — these raw recordings become the foundation for the shot list in Production. Research is where you learn the material and capture the first pass of footage simultaneously.
📥 Input
  • Learning objectives & module breakdown
  • Technical documentation & specs
  • Existing tutorials & references
  • Tooling & platform constraints
📤 Output
  • Draft screen recordings & walkthrough takes
  • Images, diagrams & infographics
  • Music, voiceover & SFX assets
  • Reference videos & B-roll clips
  • Research notes & citations
🛠 Tools Used
🤖 Claude 🎥 OBS Studio 🖼 fal.ai (image-gen) 📁 Google Drive 🔍 Perplexity 🎤 ElevenLabs 📺 YouTube
→ Open Research Hub
⬇
📋
04 Outline Pre-Production
Structure the course into modules and sections. Define the logical flow, timing estimates, and key learning points for each segment.
📥 Input
  • Learning objectives & scope
  • Research findings & assets
  • Timing constraints (e.g. 15-min videos)
  • Prerequisite knowledge map
📤 Output
  • Module-by-module outline
  • Section breakdown with time estimates
  • Key concepts & demonstrations per section
  • Asset requirements list
🛠 Tools Used
🤖 Claude 📝 Markdown 🪨 Obsidian 📄 Google Docs
→ Open Outline page
⬇
🎬
05 Script Pre-Production
Write the full narration script for each video. Include on-screen visual cues, screen recording directions, and timing annotations for the voiceover.
📥 Input
  • Course outline with section structure
  • Technical content to explain
  • Visual asset references
  • Speaking style & tone guidelines
📤 Output
  • Full narration script per video
  • Visual cues & screen direction notes
  • Timing annotations (seconds per section)
  • Voiceover recording guide
🛠 Tools Used
🤖 Claude 📝 Markdown 📄 Google Docs
→ Open Script page
⬇
🎯
06 IVQ (InVideo Questions) Pre-Production
Design interactive quiz questions embedded at key checkpoints within each video. Reinforce learning, test comprehension, and keep learners engaged.
📥 Input
  • Script with section boundaries
  • Learning objectives per module
  • Key concepts to reinforce
  • Question format preferences
📤 Output
  • Question bank per module
  • Timestamps & placement in video
  • Answer explanations & references
  • IVQ HTML page for review
🛠 Tools Used
🤖 Claude 🎯 Custom IVQ HTML 🗄️ Supabase
→ Open IVQ Manager
⬇
📸
07 Shot List & Assets Production
Translate the script into a detailed shot-by-shot plan. Specify screen recordings, talking head segments, diagrams, B-roll, and all visual assets needed for each scene.
📥 Input
  • Narration script with visual cues
  • Research assets (images, video clips)
  • Outline & timing specifications
  • Brand & style guide
📤 Output
  • Numbered shot list per scene
  • Asset generation tasks (diagrams, graphics)
  • Screen recording script & steps
  • Recording environment checklist
🛠 Tools Used
🤖 Claude 📸 Custom Shotlist HTML 🖼 fal.ai (image-gen) 🗄️ Supabase
→ Open Shot List
⬇
🎥
08 Footage Mapping Production
Map the raw recorded footage (screen captures, talking head, B-roll) to each shot in the shot list. Identify what has been captured, what is missing, and what needs re-recording.
📥 Input
  • Shot list with expected takes
  • Raw screen recordings & video files
  • Voiceover audio files
  • Status tracker (recorded / pending)
📤 Output
  • Footage-to-shot mapping matrix
  • Missing footage gap report
  • Retake & pickup list
  • File naming & folder organization
🛠 Tools Used
🎥 OBS Studio 🎞 Custom Mapping HTML 📁 Google Drive 🎨 Canva
→ Open Footage Mapping
⬇
✂️
09 Edit List Post-Production
Create the editing decision list (EDL) specifying exactly how each scene is assembled: cuts, transitions, overlays, lower thirds, audio levels, and effects.
📥 Input
  • Footage mapping matrix
  • Shot list & script references
  • Lower thirds & overlay assets
  • Audio tracks (voiceover, music, SFX)
📤 Output
  • Scene-by-scene EDL with timestamps
  • Transition & effect specifications
  • Audio mix & level instructions
  • Rendered video file per module
🛠 Tools Used
✂️ Custom EDL HTML 🎬 DaVinci Resolve 🎚 Audacity
→ Open Edit List
⬇
📺
10 Course Playlist Publishing
Upload final videos to YouTube and organize them into a structured course playlist. Add descriptions, timestamps, and end-screen elements for seamless navigation.
📥 Input
  • Rendered video files (per module)
  • Video descriptions & SEO metadata
  • Chapter timestamps per video
  • YouTube channel access & permissions
📤 Output
  • Published videos on YouTube
  • Ordered course playlist
  • End-screen & card annotations
  • Video descriptions with links
🛠 Tools Used
📺 YouTube Studio 🤖 Claude (descriptions/SEO)
→ View Course Playlist
⬇
🖼️
11 Thumbnail Publishing
Design compelling video thumbnails that drive clicks while accurately representing the content. Follow brand guidelines and thumbnail best practices (contrast, text, faces).
📥 Input
  • Video title & key concept
  • Brand style guide & colors
  • Screenshot candidates from video
  • Thumbnail template & design assets
📤 Output
  • Custom thumbnail image (1280×720)
  • Text overlay & branding elements
  • Uploaded thumbnail per video
  • A/B test results (if applicable)
🛠 Tools Used
🖼 Custom Image Generator 🖼 fal.ai (image-gen) 🎨 Canva
→ Open Image Generator
