# 🎥 production — Production Environment Pages

> **Purpose:** All production environment HTML pages organized by lifecycle phase: pre-production, production (active), post-production, and publish.

## Subdirectories

| Directory | Purpose |
|-----------|---------|
| `preprod/` | Pre-production phase — scripts, sanity checks, edit tools |
| `prod/` | Active production — readiness plans, checklists |
| `postprod/` | Post-production — review, business plan, module plans |
| `publish/` | Published/final pages — membership page |

## Rules
- Keep each phase's assets in its own subdirectory
- Link between phases using `navigation_config.json`
