☁️ Azure Blob Storage

Production assets (generated audio, images, reversal clips, references) are uploaded to Azure Blob Storage. No client-side configuration is needed — the server holds the storage connection string (from Azure Key Vault) and signs short-lived SAS tokens for every upload and read.
