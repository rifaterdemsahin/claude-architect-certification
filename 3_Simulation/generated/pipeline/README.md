# 📁 Pipeline Generated Assets

## 🎯 Purpose
This folder contains high-fidelity visual assets representing each stage of the 11-step production pipeline. These images are used in `5_Symbols/pipeline.html` to provide a visual walkthrough of the course creation process.

## 📂 Files
| File | Phase | Stage | Description |
|------|-------|-------|-------------|
| `01_problem_pipeline.png` | Pre-Production | ❓ Problem | Articulation of the core problem |
| `02_solution_pipeline.png` | Pre-Production | 🏠 Solution | Course concept and learning objectives |
| `03_research_pipeline.png` | Pre-Production | 🔬 Research | Deep-dive and draft screen recordings |
| `04_outline_pipeline.png` | Pre-Production | 📋 Outline | Module and section breakdown |
| `05_script_pipeline.png` | Pre-Production | 🎬 Script | Full narration and visual cues |
| `06_ivq_pipeline.png` | Pre-Production | 🎯 IVQ | InVideo Question design |
| `07_shotlist_pipeline.png` | Production | 📸 Shot List | Detailed shot-by-shot plan |
| `08_footage_mapping_pipeline.png` | Production | 🎥 Mapping | Recorded footage to shot matrix |
| `09_edit_list_pipeline.png` | Post-Production | ✂️ Edit List | Editing decision list (EDL) |
| `10_course_playlist_pipeline.png` | Publishing | 📺 Playlist | YouTube organization and metadata |
| `11_thumbnail_pipeline.png` | Publishing | 🖼️ Thumbnail | Clickable video thumbnail design |

## ⚙️ Rules
- **Naming Convention:** All files MUST follow the `[01-11]_[phase]_pipeline.png` format.
- **Sync:** Any changes to files here must be reflected in the mapping inside `5_Symbols/pipeline.html`.
- **Modals:** These images are designed to be viewed in a full-screen modal preview on the pipeline page.
